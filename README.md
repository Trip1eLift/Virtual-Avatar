# tcp-socket-practice
This is a POC project to use python socket server and react socket client

## Python Server
To setup python virtual environment
```console
cd server
python3 -m venv ./venv
```

To activate
```console
where.exe python
./venv/Scripts/activate
where.exe python
```

Operation in venv
```console
pip list
py -m pip install -r requirements.txt
```

```console
deactivate
```

## React Client
To setup react environment
```console
cd client
npm install
```

To run
```console
yarn start
```
